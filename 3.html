The CSS Working Group has published CSS Fonts Module Level 3 as a W3C Recommendation.
This CSS Module describes how to specify fonts used in CSS, including Web Fonts downloaded on demand.

It also describes how to access advanced typographic features in fonts, and how to control font loading. This specification is implemented in all modern browsers
